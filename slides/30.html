Developing a Web Application

Developing a web application and working as teams are always a complicated task.

Working in teams always requires harmony and coordination, this is very eminent in any kind of programing project.

Larger teams may have friction between them since there is not only one true way of doing something while programming.

Putting this aside, it is always been a seperate task to deploy the application itself as a end product.

Thankfully while web development was advancing by itself, the tools that go along with it to ensure quality of code and deploy the application was also sharpened.

These developer operation tools enabled to automate the process of ensuring code quality and deployment.
